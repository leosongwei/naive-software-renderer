naive-software-renderer
=======================

Stay simple, stay naive. (Made with lisp)

<img src="bunny.gif" />

Require:

* SBCL, QuickLisp
* cmake, make, gcc
* SDL2

How to:

* `cd` into `cutils`, `$ cmake CMakeLists.txt`, `make`
* `cd` into source code directory, `$ sbcl --load example.lisp`
